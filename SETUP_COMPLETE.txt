🐛 TROUBLESHOOTING
================================================================================

Issue: "Module not found"
Solution: pip install -r requirements.txt

Issue: "Can't find client_secret.json"
Solution: Ensure file is in same directory as GUI

Issue: "Metadata generation fails"
Solution: Check GEMINI_API_KEY in .env file

Issue: "No videos found"
Solution: Ensure you have draft (private) videos

For more help: See GUI_README.md "Troubleshooting" section
