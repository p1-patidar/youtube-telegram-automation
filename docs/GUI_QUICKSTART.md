# GUI Quick Start Guide

## 🚀 Get Started in 5 Minutes

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Add Your API Keys
Create a `.env` file in the same directory:
```
GEMINI_API_KEY=your_gemini_api_key_here
```

### Step 3: Setup Google OAuth
- Go to https://console.cloud.google.com
- Create a new project
- Enable YouTube Data API v3
- Create OAuth 2.0 credentials (Desktop)
- Download and save as `client_secret.json`

### Step 4: Launch the GUI
```bash
python run_gui.py
```

---

## 📋 Basic Workflow

### Adding Your First Account
1. Click **"Add New Account"**
2. Enter account name (e.g., "personal")
3. Browser opens for Google login
4. Grant permissions
5. Done! Account is ready

### Scheduling Videos
1. Click **"Run Automation"**
2. **Step 1**: Review unscheduled videos
3. **Step 2**: 
   - Pick start date
   - Set videos per day (e.g., 2)
   - Set time slots (e.g., 09:00, 15:00)
4. **Step 3**: Click "Start Processing"
5. Watch videos get scheduled in real-time!

### Managing Your Data
1. Click **"Manage Accounts"**
2. Go to **"Data Management"** tab
3. Select your account
4. Click **"View Summary"** to see statistics
5. Use **"Cleanup Old Data"** to remove old entries

## ⚙️ Common Settings

### Recommended Time Slots
```
1 video/day:   09:00
2 videos/day:  09:00, 15:00
3 videos/day:  09:00, 13:00, 18:00
```

### Date Format
Always use: **DD-MM-YYYY**
- Example: 10-11-2025 (November 10, 2025)

### Video Per Day
- Start with 1-2 videos
- Increase based on your schedule

## 🐛 Quick Troubleshooting

**"No videos found"**
→ Make sure you have draft (private) videos

**"Authentication failed"**
→ Check client_secret.json exists

**"Metadata error"**
→ Verify GEMINI_API_KEY in .env file

**"Button won't click"**
→ Wait for previous operation to complete

✅ **Saved Automatically:**

📁 **Saved In:**
- `processed_videos_data/` - Video records
- `youtube_accounts/` - Account tokens

## 💡 Pro Tips

1. **Test with 1 video first** - Run on single video to test
2. **Schedule off-peak hours** - YouTube gets less traffic at specific times
3. **Use consistent time slots** - Helps audience know when to expect new videos
4. **Monitor your quota** - Check YouTube API usage regularly
5. **Keep backups** - Save your `processed_videos_data/` folder
